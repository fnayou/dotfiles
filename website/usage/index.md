# Usage

This section is about **using** the setup day to day — once the packages you want are installed. For
getting things onto a machine, see [Installation](../installation.md); this section assumes that's done.

| Page | What it covers |
|---|---|
| [Daily Workflow](daily-workflow.md) | A typical terminal session with the shell, Git, and task helpers |
| [Aliases](aliases.md) | The shell aliases the config defines, grouped by purpose |
| [Functions](functions.md) | The shell functions worth knowing about |
| [Taskfile](taskfile.md) | The `task` command entrypoint — safe vs. system-modifying tasks |

!!! note "Take what fits"
    None of this is mandatory. The aliases and helpers are conveniences — adopt the ones that match how
    you already work and ignore the rest.
