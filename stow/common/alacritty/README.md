# alacritty

Managed [Alacritty](https://alacritty.org/) terminal configuration. Stows into `~/.config/alacritty/`.

## What it configures

- Window, font, keyboard bindings, and shell.
- Catppuccin Macchiato color theme, imported from a separate theme file.

## Files

| File | Stowed to | Purpose |
|---|---|---|
| `.config/alacritty/alacritty.toml` | `~/.config/alacritty/alacritty.toml` | Main config — window, font, key bindings, shell, theme import |
| `.config/alacritty/catppuccin-macchiato.toml` | `~/.config/alacritty/catppuccin-macchiato.toml` | Catppuccin Macchiato color theme |
