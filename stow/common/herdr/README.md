# herdr

Managed [Herdr](https://herdr.dev/) agent-multiplexer configuration. Stows into `~/.config/herdr/`.

Source: [github.com/ogulcancelik/herdr](https://github.com/ogulcancelik/herdr).

Herdr loads `~/.config/herdr/config.toml` automatically on startup — no activation step beyond stowing.

## What it configures

- Theme, terminal, UI, and toast settings.

## Files

| File | Stowed to | Purpose |
|---|---|---|
| `.config/herdr/config.toml` | `~/.config/herdr/config.toml` | Main Herdr configuration |
